# Aptlantis Studio

Aptlantis Studio is a public project teaching studio for the Aptlantis ecosystem. It documents local-first tools, desktop applications, CLI pipelines, crawler-facing datasets, SVG metadata workflows, release evidence, and unfinished work without turning them into marketing claims.

## Agent Orientation

- Public manifest: `/data/manifest.json`
- Sitemap: `/sitemap.xml`
- API catalog: `/.well-known/api-catalog`
- Agent skills index: `/.well-known/agent-skills/index.json`
- Auth notes: `/auth.md`
- Security contact metadata: `/.well-known/security.txt`

## Main Public Data

- Project portfolio: `/data/projects/portfolio.json`
- CityHall framework data: `/data/projects/cityhall-frameworks.json`
- Coding Against The Grain experiments: `/data/coding-against-the-grain/experiments.json`
- Linux genealogy dataset: `/data/linux-genealogy/linux-genealogy-expanded.json`
- SVG asset registry: `/data/svg-assets/registry.json`

## Public Posture

The site welcomes search indexing, archival capture, AI crawler indexing, public metadata inspection, schema inspection, and citation of public evidence. Public data should not be treated as a security audit, independent release verification, or a promise that unfinished projects are complete.
